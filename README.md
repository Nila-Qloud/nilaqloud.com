# nilaqloud.com

Website for [Nila Qloud](https://nilaqloud.com) — developer infrastructure for modern development.

## Sites

| Site | Domain | Tech |
|---|---|---|
| Landing page | [nilaqloud.com](https://nilaqloud.com) | Static HTML/CSS |
| Tunnel docs | [tunnel.nilaqloud.com](https://tunnel.nilaqloud.com) | Astro + Starlight |

## Development

```bash
# Landing page — just open the file
open nila-landing/index.html

# Docs site
cd nila-tunnel-docs
npm install
npm run dev          # http://localhost:4321
```

## Deploy

Push to master — Cloudflare Pages auto-deploys both sites.
